# Changelog

## 0.1.0-preview.1

- add a C# 14 binding to the stable Tesseract 5 C API;
- add safe Gray8, RGB24, and RGBA32 pixel ownership;
- add deterministic native discovery and diagnostics;
- add engine pooling and ordered, bounded batch recognition;
- add explicit candidate-language evaluation and OSD access;
- add optional ImageSharp 3 and SkiaSharp 4 adapters;
- add reproducible Tesseract 5.5.2 builds for six desktop RIDs;
- add native OCR integration, conformance, security, and packaging workflows.
